APLICACIÓN MOVIL -
NOTICIAS MOON

Resumen del Proyecto:
Este proyecto desarrolla una aplicación móvil con un inicio de sesión y una pantalla principal que presenta varios noticieros, proporcionando acceso inmediato a contenido informativo.

La autenticación asegura una experiencia personalizada, mientras que la interfaz organiza los noticieros de forma clara, facilitando el acceso a las noticias.

APLICACIÓN WEB - INICIO DE SESIÓN SENCILLO

Resumen del Proyecto:
Este proyecto consiste en un sistema de inicio de sesión básico desarrollado con Apps Script y Google Sheets.

Utiliza Apps Script para gestionar la autenticación de usuarios y Google Sheets para almacenar y verificar las credenciales, proporcionando un método simple y efectivo para acceder a la aplicación.

DESARROLLO DE SOFTWARE - GESTIÓN DE INVENTARIO

Resumen del Proyecto:
Este proyecto es una aplicación de escritorio diseñada para gestionar inventarios, proporciona herramientas para optimizar la administración de articulos.

Permite controlar y supervisar los niveles de productos, realizar seguimiento de entradas y salidas, y optimizar la administración de stock para mejorar la eficiencia operativa.

AUTOMATIZACIÓN DE INVENTARIO - Ingeniería de Software

Resumen del Proyecto:
Este proyecto desarrolla un modelo de inventario para una tienda en Zipaquirá, buscando mejorar su gestión mediante observaciones y encuestas, identificando oportunidades para optimizar el control de productos y supervisar el sistema.

Se utilizó el lenguaje unificado de modelado para visualizar, construir y documentar el sistema, mejorando así la gestión automatizada y la organización para evitar pérdidas significativas.

Ver archivo

INFORME DE CONTROL FINANCIERO - Empresa INNOVATECH

Resumen del Proyecto:
Este archivo ofrece un análisis detallado de las finanzas de la empresa, cubriendo la nómina, el estado financiero, las horas extra, la depreciación de activos, los elementos del costo y el punto de equilibrio.

También se incluyen los estados de costos y resultados, proporcionando una visión completa para la evaluación del rendimiento financiero y la planificación estratégica.

Ver archivo

TABLERO DE POWER BI:
Información de Contacto

Resumen del Proyecto:
Este archivo presenta un tablero de Power BI que organiza y visualiza una lista de nombres y direcciones.

El tablero facilita la consulta y el análisis de datos de contacto, permitiendo una gestión más eficiente y una visualización clara de la información relevante.
